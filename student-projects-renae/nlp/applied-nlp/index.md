- [RAG Question Answering for Board Games](/student-projects/nlp/applied-nlp/rag-boardgame-qa/) — Uses retrieval-augmented generation to answer board game questions.
- [Code Documentation Generator](/student-projects/nlp/applied-nlp/code-documentation-generator/) — Automatically generates documentation from source code.
