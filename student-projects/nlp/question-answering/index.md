This page collects NLP projects in the **Question Answering** category.

## Projects

- [Knowledge Base Question Answering (RAG)](/student-projects/nlp/question-answering/knowledge-base-qa/) — A retrieval-augmented question answering system built on SQuAD using TF-IDF and T5.
- [Fitness FAQ Question Answering](/student-projects/nlp/question-answering/fitness-faq-qa/) — A domain-specific QA system for answering fitness-related questions.
- [RAG Board Game Assistant](/student-projects/nlp/question-answering/rag-boardgame-qa/) — A retrieval-based system for answering questions about board games.
- [RAG Game Assistant](/student-projects/nlp/question-answering/rag-game-assistant/) — A question answering system for game-related knowledge using retrieval and generation.
