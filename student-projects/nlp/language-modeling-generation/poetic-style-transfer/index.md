## Overview

This project focuses on transfers poetic writing styles between texts.

The primary method was fine-tuning the pre-trained model BART with a manually created dataset consisting of 15 paired examples of Frost poems and their Shakespearean-style translations.

## Data and System Design

The project draws on Project Gutenberg, X, Hugging Face Dataset.

The core system is built around BERT, T5-small, T5, RAG. The workflow is designed to move from raw text to a structured or interpretable output with as little manual intervention as possible.

The implementation centers on pretrained models and standard Python NLP tooling.

## Results

The project reports qualitative or comparative results rather than a single headline metric. The results point to a workable solution for the target task and a clear path for refinement.

## Why This Matters

Generation projects are useful because they show how language models can move from pattern recognition to actual output creation. This project also makes clear that dataset design and prompting choices shape the final result.
