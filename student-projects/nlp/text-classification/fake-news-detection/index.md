## Overview

This project focuses on identifies misinformation in news and online articles.

Natural Language Processing Final Project Guidelines and Suggestions Rules: • All projects are individual projects. No paired or team projects.

---

## Data and Modeling

The project draws on Amazon, SARC, X.

The core system is built around BERT, GPT-2, LSTM, RAG. The workflow is designed to move from raw text to a structured or interpretable output with as little manual intervention as possible.

The implementation centers on pretrained models and standard Python NLP tooling.

---

## Results

The project reports qualitative or comparative results rather than a single headline metric. The results point to a workable solution for the target task and a clear path for refinement.

---

## Why This Matters

Classification projects like this matter because they turn large collections of text into signals that can actually be used. That can support moderation, decision-making, or domain analysis depending on the setting.
