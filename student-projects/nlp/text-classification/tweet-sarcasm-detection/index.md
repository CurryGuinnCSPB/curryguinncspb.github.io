This project focuses on detects sarcastic tweets using deep learning models.

## Data and Modeling

The project draws on SARC, X.

The core system is built around BERT, spaCy, TF-IDF, Logistic Regression. The workflow is designed to move from raw text to a structured or interpretable output with as little manual intervention as possible.

The implementation centers on pretrained models and standard Python NLP tooling.

Classification projects like this matter because they turn large collections of text into signals that can actually be used. That can support moderation, decision-making, or domain analysis depending on the setting.
