## Overview

This project focuses on classifies customer sentiment in Amazon musical instrument reviews.

NLP Final Project Presentation Sentiment Analysis: Amazon Reviews of Musical Instruments Nate Bliton nabl2290 Fall 2025 CSPB 3832 Natural Language Processing Natural Language Processing (NLP) Task: Sentiment Analysis Sentiment analysis is interesting to me, the ability of a trained model to be able to evaluate whether a body of text is "positive" or "negative". We can work with an association between a body of text and a sentiment, train a model on that association, and use that model to predict the sentiment of other similar text.

---

## Data and Modeling

The project draws on Amazon, X.

The core system is built around TF-IDF, Logistic Regression, Naive Bayes. The workflow is designed to move from raw text to a structured or interpretable output with as little manual intervention as possible.

The implementation centers on pretrained models and standard Python NLP tooling.

Classification projects like this matter because they turn large collections of text into signals that can actually be used. That can support moderation, decision-making, or domain analysis depending on the setting.
